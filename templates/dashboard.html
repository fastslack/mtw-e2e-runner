Tools
▤ Module Analysis
Detects action sequences that repeat across 2+ tests — the canonical extraction candidates for reusable $use modules. Also surfaces unused modules and adoption counts. Pair with the test-improver agent (Claude Code) for AI-driven refactoring using the copied prompt.
Pick a project and click Run analysis.
📷 Capture URL
Open any URL in a pooled browser and save a screenshot. Auth headers from the project config are applied automatically.
🔍 Analyze Page
Inspect a URL and return all interactive elements (buttons, links, inputs, forms, tabs) with their CSS selectors plus suggested test scaffolds.
⚔ Verify Issue
Fetch a GitHub/GitLab issue, generate matching E2E tests, run them, and produce a verification report. Generation needs ANTHROPIC_API_KEY; without a key, you get a prompt to paste into Claude Code.
★ AI Agents (Claude Code)
Spawning Claude Code agents from the dashboard isn't possible directly. Use these one-click prompts to paste into your Claude Code session and trigger the right agent.
test-improver Refactor verbose tests, extract modules, harden selectors, kill flakes.
test-creator Explore a page, design action sequences, write a new test from scratch.
test-analyzer Diagnose failures, flaky patterns, network errors — read learnings + screenshots.
